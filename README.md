# Vector

## Description
A template container is an analog of std::vector. It has methods similar to the standard std::vector:
* begin, end, cbegin, cend, rbegin, rend, crbegin, crend
* PushBack, EmplaceBack, Emplace, Insert
* PopBack, Erase, Clear
* Reserve, Resize
* Size, Capacity, Empty
* Back, Front, operator[]
* Swap

## Features
* provides a strict security guarantee
* efficient storage
* using the placement new operator
* using move semantics

### Using technologies
C++, STL, RAII, move semantics


### System requirements
C++ 17
